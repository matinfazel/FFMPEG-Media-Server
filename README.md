# ffmpeg-pusher
Use ffmpeg push stream like rtsp rtmp
In this program input can be in rtsp,stmp,mp4,... format 
# Installation
`git clone https://git.psvira.com/Fazel/ffmpeg.git`<br/>
`cd ffmpeg`<br/>
`mkdir build`<br/>
`cd build`<br/>
`cmake ..`<br/>
`make`<br/>
`./test input_stream output_stream`<br/>
